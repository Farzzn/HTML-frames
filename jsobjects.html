JavaScript Objects





This example shows the usage of objects in java script. Here I have declared three objects person1, person2 and person3. All of these objects have firstname, lastname, address and a function showdetials(). Through this function we can get detials of each person using "this" keyword.

for example,
"this.firstname" will give first name of the object.
