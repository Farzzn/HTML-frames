BOX MODEL IN CSS




The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content. In CSS box model, each element is represented as a box with the following parts or properties:
Content:It is the area where the content of the box is displayed
Padding:The space or area between the outer edge of the content box and the inner edge of the border
Border:The area between margin and the padding; its width, style and color can be changed
Margin: The area or space outside the border or outside the CSS box; it is transparent

To get the total width or height of a CSS Box we have to use the following formulas.
CSS Box's width= left border's width + left padding's width + content box's width + right padding's width + right border's width
CSS Box's height = top border's height + top padding's height + content box's height + bottom padding's height + bottom border's height
For example, if a element div has the following CSS declarations
          width: 900px;
          height: 600px;
          padding: 20px;
          border: 5px
its total width/height would then be 950px/650px:
Width and height CSS properties only set width and height of the content box not the entire CSS box.
